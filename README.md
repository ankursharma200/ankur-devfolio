# 🚀 Ankur Sharma — Developer Portfolio

A modern, responsive portfolio website showcasing my work experience, projects, skills, and achievements.

🔗 **Live:** [Coming Soon](#)

---

## ✨ Features

- 🌙 Dark/Light mode toggle
- ⌨️ Typewriter animation on hero section
- 📱 Fully responsive (mobile-first)
- 📧 Contact form with EmailJS integration
- 🔔 Resume view tracking — get notified when someone views your resume
- 🎨 Smooth animations with Framer Motion
- ⚡ Fast build with Vite

---

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/ankursharma200/ankur-devfolio.git
cd ankur-devfolio

# Install dependencies
npm install

# Run development server
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in the root:

```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_contact_template_id
VITE_EMAILJS_RESUME_TEMPLATE_ID=your_resume_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

---

## 📦 Build for Production

```bash
npm run build
```

Output goes to the `dist/` folder.
